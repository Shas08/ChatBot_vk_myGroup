# ChatBot_vk_myGroup
Чат бот для моей небольшой группы вк для сдачи лабораторной работы

Чат-бот может отправить фото, если выбрать последовательность команд: "Начать -> Расписание -> {Курс} -> {Группа}", для этого нужно создать папку расписание в директории с кодом и залить туда фотографии формата jpg.

P.S. Т.к. этот чат-бот написан для закрытой группы ВК по интересам (студенты определённого ВУЗа, факультета, направления), то токен я вырезал, протестировать бота могу только я.
